As linguagens mais utlizadas para WebSite



HTML

O HTML é uma linguagem de marcação, que serve de base para a criação de sites.
Com o HTML, é possível a criação, exibição, formatação e editação de elementos de uma página de internet, como:
  • Textos
  • Imagen
  • Formulários
  • Tabelas



JavaScript

A linguagem de programação JavaScript é usada para rodar scripts através dos navegadores.
Com o JavaScript, algumas aplicações podem ser rodadas sem a necessidade de haver trocar de informações com o servidor.
Assim, um documento poderia ser alterado através destes scripts que interagem diretamente com o usuário.
Iremos aprender JavaScript logo após estudar HTML.


PHP


| A linguagem de programação PHP é do tipo interpretada e é, atualmente, uma das mais usadas no mundo.
O PHP é vastamente usado para criar aplicações e sistemas, do lado do servidor.
Usa-se muito o PHP para criar páginas com conteúdo de uma maneira dinâmica, além de ser possível acessar
e interagir com servidores e banco de dados.
